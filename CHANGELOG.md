# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-26

### Added

- Initial browser-first web search library core (`SearchClient`, provider abstraction).
- TF-IDF and cosine-similarity ranking with deterministic tie-break behavior.
- Built-in fetch-based provider and DuckDuckGo provider factory.
- Typed error model for validation, network, timeout, abort, and provider failures.
- Test suite split into unit, integration, and property-based coverage.
- GitHub Actions CI pipeline with changed-file detection and npm publish automation.

### Changed

- License confirmed as MIT.
- README and project docs expanded with installation, architecture, and development guidance.
